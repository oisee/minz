# simple_add

## Files
- **simple_add.minz** - Original MinZ source code
- **simple_add.mir** - MinZ Intermediate Representation (shows optimization passes)
- **simple_add.a80** - Generated Z80 assembly with SMC

## Features Demonstrated
- Self-Modifying Code (SMC) functions

## SMC Parameters
```asm
add_param_a EQU add + 1
add_param_b EQU add + 4
```
