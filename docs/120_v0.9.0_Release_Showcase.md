**Traditional Approach** (145 T-states for 5 chars):


### 3. Length-Prefixed String Architecture

```asm
; Old way (null-terminated):

; MinZ way (length-prefixed):
string: DB 5,"Hello"        ; 6 bytes, O(1) length!
```
