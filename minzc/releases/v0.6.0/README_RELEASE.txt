MinZ v0.6.0 "Pattern Matching Revolution" Release
================================================

This archive contains:

1. minzc - The MinZ compiler binary (macOS)
2. RELEASE_NOTES.md - Detailed release notes
3. 041_Pattern_Matching_Revolution.md - Technical article
4. examples/ - State machine examples demonstrating pattern matching
5. *.a80 - Compiled assembly files ready for Z80

To use:
1. Extract the archive
2. Make minzc executable: chmod +x minzc
3. Compile MinZ files: ./minzc input.minz -o output.a80

For full documentation, see:
https://github.com/oisee/minz

This release introduces revolutionary pattern matching that transforms
how we write state machines and control flow on Z80 hardware!
